# Web

## Prerequisites

- Node.js >=20

## Install

`npm install`

## Environment

Copy `.env.example` to `.env.local`.

## Dev server

`npm run dev` -> http://localhost:3000

## Build

`NEXT_PUBLIC_API_URL=... npm run build`

## Quality checks

`npm run lint`

`npx tsc --noEmit`

`npm run format:check`
