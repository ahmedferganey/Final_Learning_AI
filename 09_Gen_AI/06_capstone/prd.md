# Product Requirements Document — AI Productivity Platform

## Problem
Users need a single AI platform that can:
- answer questions
- retrieve grounded knowledge
- use tools
- expose APIs
- be measured and improved over time

## Core Components
- Gateway API
- Chat Service
- RAG Service
- Agent Service
- Eval Service
- Ingestion Worker

## Primary User Flows
1. Ask a normal chat question
2. Ask a document-grounded question
3. Ask the system to use tools
4. Benchmark outputs
5. Review logs and metrics

## Non-functional Requirements
- typed APIs
- modular services
- documented contracts
- local development flow
- test coverage
